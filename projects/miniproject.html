Smart Slot Assignment for Urban EV Charging

Overview

The project focuses on developing a Smart Slot Assignment System for Urban EV Charging to address challenges caused by the increasing adoption of electric vehicles and the limited availability of charging infrastructure in urban areas.

solution includes:

  • An online slot-booking website for EV charging.
  • Intelligent algorithms to optimize slot usage.
  • Integration of map-based visualization of charging stations.
  • Enhanced user experience, reduced waiting times, and improved station utilization

System Architecture

Tech Stack

    • Languages: Python, HTML, CSS, JavaScript
    • Database Management: MySQL
    • Framework & Libraries: Flask, Geocoder, Folium Map API

My Contributions

  • Integrated Folium Map API to display charging stations with live status and emergency slot availability
  • Designed and implemented dynamic query generation algorithms to retrieve nearby charging stations
  • Developed the user interface to enable easy booking and information display

Conclusion

The smart EV slot booking system, enhanced with an emergency slot feature, successfully improves charging station utilization and revenue.

By combining data-driven insights, intelligent algorithms, and an interactive user interface, the system Reduces waiting times, Optimizes resource allocation, Supports sustainable urban mobility. This solution demonstrates an effective approach to modern EV charging challenges in urban environments.
